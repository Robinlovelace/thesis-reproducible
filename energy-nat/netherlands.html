Loading and plotting Dutch commuting data

As with the Enlgish commuting data, we begin by loading and inspecting the raw commuting data. As we shall see, it is in a different format than the English data, so needs to be treated differently. In addition, we need to translate some of the Dutch words into English, using the command gsub. As with the English data, the information was taken from an official data dissemination portal, in this case the CBS. The data is publicly available on the internet.
